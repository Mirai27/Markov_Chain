Необходимо установить отсутсвующие зависимости
pip install random
pip install pymorphy3
pip install re

Текст использован научный (энергетика). Находится он в файле text.txt

После запуска программы необходимо подождать обработки текста.
Затем будет непрерывное приглашение от программы на ввод пользователя
Если для введенной цепочки слов не удается найти подсказку программа убирает по одному слову с начала,
до тех пор пока подсказка не будет найдена, иначе выводится сообщение о том, что подсказок нет.
Для завершения работы необходимо нажать Ctrl + C
